Resolution of the Tenten machine

July 4, 2020 - 5 minute read -
HackTheBox

Tenten is a medium level machine based on Linux, the initial approach is heavily based on web enumeration. The entire machine has a CTF style that requires some guessing, steganography and avoid rabbit holes. The privilege escalation process is pretty easy.

Exploiting - Vulnerable WordPress plugins

There’s an interesting vulnerable WP plugin (CVE-2015-6668) with a PoC on GitHub: CV filename disclosure on Job-Manager WP Plugin

This exploit uses a vulnerability named IDOR and a technique named fuzzing to locate the hosted file located at the WP uploads using a range of dates and some file extensions.
