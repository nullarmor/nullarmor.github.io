Run the program and send an input with a recognizable number to search using the gef module named search-pattern.

So now we got two interesting memory addresses, 0x7fffffffe4f8 and 0x7fffffffe460, to get the space between them we just need to subtract them:

The result is 0x98 or 152 in decimal. So now we know how many characters before crash the program through the input.
