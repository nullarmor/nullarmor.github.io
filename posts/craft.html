Craft - Hack The Box

Resolution of the Craft machine

June 28, 2020 - 7 minute read -
HackTheBox

Craft is a medium level machine based on Linux, the enumeration is a important component at every stages, also Python knowledge is essential. The initial approach requires code review skills in order to identify a Python injection point in the API. The privilege escalation is based on docker, enumeration and how a determined vault program works.

There’s a file named dbtest.py that connects to a MySQL database, but the credentials are actually loaded from the settings.py file, this file is not included in the repository for security reasons.

Again, after some time searching for more valuable information, we found a suspicious issue, the dinesh user says that there’s a unsanitized parameter being inserted into the database, so he, himself, will take care of this by fixing this vulnerability.

Doing a code review and looking for the supposed fix, we found that he added a worse vulnerability by mistake, he used an eval() function in a attempt to handle the problem, but this eval receives the user input directly.

A malicious user can send a crafted request and execute system commands through the eval function.
