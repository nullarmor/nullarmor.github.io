Blocky - Hack The Box

Resolution of the Blocky machine

June 30, 2020 - 4 minute read -
HackTheBox

Blocky is an easy level machine based on Linux, the initial approach is based on enumeration. Has two ways to get into the server and two ways to escalate to root user during the post exploitation. The privilege escalation process is pretty fast and doesn’t require much time.

Post Exploitation - Escalating privileges through a kernel exploit (2nd method)

Doing a basic enumeration, we noticed that the server’s kernel version is vulnerable, this vulnerability has the CVE 2017-6074.

Linux Kernel 4.4.0 (Ubuntu) - DCCP Double-Free Privilege Escalation

After compile the exploit locally, send it to the machine through Python SimpleHTTPServer and execute, we easily got a root shell!
