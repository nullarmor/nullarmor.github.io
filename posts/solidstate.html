SolidState - Hack The Box

Resolution of the SolidState machine

November 7, 2020 - 4 minute read -
HackTheBox

SolidState is a medium level machine, based on Linux. The main point is based on mail management, including protocols like POP3 and SMTP. The initial exploitation is based on a vulnerability that exists in JAMES Remote Admin 2.3.2 The privilege escalation process relies on a Python script, owned by root but available to other users, so any user is able to edit that script.

Recon

Nmap reveal us some mail services running on the machine, including an interesting service named James Remote Admin 2.3.2

Exploiting - Getting an exploit with a simple recon

Googling in search for some exploits available for James Remote Admin version 2.3.2 we got the following Exploit-DB exploit:

Apache James Server 2.3.2 - Remote Command Execution

It’s a script that login using the default credentials (root:root), create an user and add a malicious command to be executed as RCE when some user receives an email.
