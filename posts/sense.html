Exploiting - Editing exploits and getting a reverse shell

After searching on Google for suitable exploits for pfSense version 2.1.3, we found an interesting one:

pfSense < 2.1.4 - ‘status_rrd_graph_img.php’ Command Injection

This exploit(CVE-2014-2688), exploits a vulnerability present in a pfSense module named status_rrd_graph_img.php, which receive some unsanitized parameters from the user. It’s possible to send a malicious payload encoded as octal, this payload will be interpreted by the server and executed as a system command.
