Shocker - Hack The Box

Resolution of the Shocker machine

July 1, 2020 - 4 minute read -
HackTheBox

Shocker is an easy level machine based on Linux, the initial approach is based on a infamous vulnerability called ShellShock. The privilege escalation is pretty straightforward and doesn’t require any special skills.

Recon

The port 80 has a page with a simple but enigmatic image:

After that, enumerating web directories using the gobuster tool give us the following output:

We noticed that there’s a folder named cgi-bin, this directory comes from a tipical vulnerability named ShellShock, where it’s possible to inject system commands through a vulnerable script inside the cgi-bin directory. Now we have to look for any script inside that directory in order to exploit it, we have to enumerate any kind of script with these extensions:

  • .cgi
  • .php
  • .py
  • .sh
  • .rb
