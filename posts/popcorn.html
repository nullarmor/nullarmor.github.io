Post Exploitation - Exploiting the Linux PAM

On privilege escalation process we easily found an interesting directory located at /home/george/.cache/ that has a file called motd.legal-displayed. Searching for any exploit for this service we found an exploit with the CVE 2010-0832.

Linux PAM 1.1.0 (Ubuntu 9.10/10.04) - MOTD File Tampering Privilege Escalation (2)

This bash script basically exploits the Linux PAM permissions of MOTD, adding a SSH key to change the root user’s password to toor.

Downloading the exploit, sending it to the Popcorn machine through the Python module called SimpleHTTPServer, spawning a better shell using pty and executing will result in a privilege escalation to root!
