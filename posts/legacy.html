Legacy - Hack The Box

Resolution of the Legacy machine

May 18, 2020 - 2 minute read -
HackTheBox

Legacy is an easy level machine based on Windows, the machine is very easy but unstable at some points. The whole machine is based on a SMB vulnerability named MS08-067 that exploits a stack buffer overflow. Legacy doesn’t require privilege escalation.

Recon

Starting with Nmap:

Looking for exploits and we found an exploit for SMB that is running on port 445. The exploit is named MS08-067, a vulnerability in the stack that allow the attacker to perform a RCE.

To check if the machine is vulnerable we can use a Nmap script named smb-vuln-ms08-067.nse

Exploiting - Exploiting the SMB

Metasploit has a module for this exploit, it’s named windows/smb/ms08_67_netapi

Exploiting - Using a custom exploit

For knowledge purposes I made a custom exploit that exploits the SMB vulnerability and spawn an interactive reverse shell to us, it’s available on my GitHub :)
