Bitlab - Hack The Box

Resolution of the Bitlab machine

June 27, 2020 - 6 minute read -
HackTheBox

Bitlab is a medium level machine based on Linux, the initial approach is heavily based on enumeration, requiring git knowledge. The privilege escalation has two stages, the first one from user www-data to clave and the second one from clave to root. This machine has docker containers that runs some services. The second stage of the privilege escalation requires a knowledge about dynamic reverse engineering on Windows executable files.
